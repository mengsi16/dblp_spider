# dblp_spider
dblp会议/期刊爬虫。
### 文件目录
```bash
USE_Selenium
├─ engine.py
├─ modifyBibName.py
├─ save.py
├─ Selenium_Get_DBLP.py
├─ UI.py
├─ UI.spec
├─ UI2.py
├─ dist
│  └─ UI.exe
get_reference.py
```
爬虫核心逻辑在Selenium_Get_DBLP.py
可以使用UI2.py打开一个简易的UI爬虫界面。
get_reference.py 可以快速筛选引用量达标的论文出来。（还没有做UI界面）
